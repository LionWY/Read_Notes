# 第二章 对象、消息、运行期
1. 面向对象语言中，对象就是“基本构造单元”（building block），开发者通过对象来存储并传递数据。
2. 对象之间传递数据并执行任务的过程就是“消息传递”（messaging）
3. 运行期环境：就是开发者编写的代码，提供了一些使得对象之间能够传递消息的重要函数了，并且包含创建类实例所用的全部逻辑
